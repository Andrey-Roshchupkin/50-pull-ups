50 pull-ups in ... weeks

Hi! In this app you can follow training program to raise number of your pull-ups. It works like ToDo list. Note that all data stores in your browser and doesn't synchronize between different devices. If you fill that you have to start from third week, when you check your first set, all previous sets will be checked automatically.

Intermediate level

To start this level you should make at least 7 pull-ups in one set. If you don't, make Australian, negative and pull-ups with help 3 times a week.

Advanced level

To start this level you should successfully complete last week of previous level.
