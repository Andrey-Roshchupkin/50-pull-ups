50 pull-ups in 7 (14) weeks

Hi! In this app you can follow training program to raise the number of your pull-ups. It works like ToDo list. Note that all data stores in your browser and doesn't synchronize between different devices. If you feal that you can start from third week, when'll you check your first set, all previous sets will be checked automatically.

Don't forget to rest between sets as much time as needed. Remember that main target for every training is to complete the entire workout.

Intermediate level

To start this level you should make at least 7 pull-ups in one set. If you don't, make Australian, negative and pull-ups with help 3 times a week.
